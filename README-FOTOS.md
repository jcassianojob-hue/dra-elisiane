# 📁 Estrutura de Arquivos – Site Dra. Elisiane Elias

## Pasta `assets/` — coloque as fotos aqui

| Nome do arquivo                                          | Onde aparece                          | Tamanho ideal |
|----------------------------------------------------------|---------------------------------------|---------------|
| `dra-elisiane-elias-pediatra-osasco-sp.jpg`             | Hero (principal) + Seção Sobre        | 480×640px     |
| `consulta-pediatrica-crianca-sao-paulo.jpg`             | Card de Pediatria                     | 400×400px     |
| `consulta-hebiatra-adolescente-osasco.jpg`              | Card de Hebiatria                     | 400×400px     |

## ⚠️ Importante
- Os nomes têm palavras-chave locais para SEO (não renomeie)
- Se as fotos locais não carregarem, o site usa imagens do Unsplash automaticamente
- Formatos aceitos: `.jpg` ou `.webp` (preferível para performance)

## 🚀 Deploy na Netlify
1. Acesse netlify.com → "Add new site" → "Deploy manually"
2. Arraste a pasta inteira do projeto
3. Pronto! O site vai ao ar em segundos.
